Generate Animation Files with Custom Resolution

This is a tool that creates an .ipv animation file with a custom resolution, which can be opened and edited in ibisPaint and ibisPaint X without the use of the prime membership

Resolutions abover 1080p may fail to be playable, exportable, uploadable, or editable on some devices

Checkout the file format reverse engineering project to figure out how this format works and if you have issues with this tool post them here
